**A self-service Tableau story analyzing 7,500+ fatal U.S. police shooting incidents — built so non-technical stakeholders can explore demographic disparities independently.**

## 🔎 What this project does

- Multi-sheet **interactive Tableau dashboard** with **LOD calculations, KPIs, and geospatial mapping** over 7,500+ incident records
- **Demographic rate and proportional analysis** identifying statistically significant use-of-force disparities across race, age, gender, and geography
- **Parameter-driven views** that let stakeholders slicUpgrade README with badges and analysis detailse by state, year, demographic group, and threat level without touching the underlying data

## 📊 Key analysis dimensions

Trends over time · demographic rates vs. population proportions · geographic hotspots by state and city · armed/unarmed status · body-camera presence · mental-illness indicators

## ▶️ How to view

Open `US Police Shooting Story.twbx` in **Tableau Desktop** or the free **[Tableau Public](https://public.tableau.com/)** app to explore the interactive story, or read the PDF deck for the summarized findings.

## 🛠️ Tools

`Tableau` (LOD expressions, parameters, story points, geospatial mapping) · `CSV/Excel`
