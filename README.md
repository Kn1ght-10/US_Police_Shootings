# 🚔 US Police Shootings Analysis

An exploratory data analysis of fatal police shootings in the United States, visualized with Tableau.

## Overview
This project examines patterns in U.S. police shooting incidents — trends over time, demographics, geography, and other contributing factors — using an interactive Tableau story.

## Contents
- `F.csv` — raw dataset of police shooting incidents
- `US Police Shooting Story.twbx` — interactive Tableau packaged workbook
- `US Police Shooting Presentation.pptx` / `.pdf` — summary slide deck of key findings

## Tools Used
- Tableau
- CSV / Excel

## How to View
Open the `.twbx` file in Tableau Desktop or Tableau Public to explore the interactive dashboard, or view the presentation files for a summarized walkthrough.
